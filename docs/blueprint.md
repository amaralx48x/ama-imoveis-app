## Core Features:

- Hero Section: A visually appealing hero section with a main title and background image.
- Property Search Bar: A search bar with filters for operation type (e.g., rent, sale), city, and property type.
- Featured Properties Carousel: A carousel showcasing featured properties.
- Agent Profile Section: A section displaying the real estate agent's photo, name, and CRECI number.
- Client Reviews Section: A section dedicated to showcasing client testimonials and reviews.
- Lead Generation Form: A contact form for generating leads (messages).
- Property Details Page: A page dedicated to displaying an Image gallery, property price, area, rooms, and a form to schedule visit for a specified property based on the [imovelId] parameter in the /imovel/[imovelId] route.

## Style Guidelines:

- Primary color: Gradient from vibrant pink (#FF69B4) to deep violet (#8A2BE2) to reflect the brand colors and add a modern touch.
- Background color: Very dark gray (#121212) to ensure excellent contrast.
- Accent color: A lighter shade of violet (#9400D3) used for highlighting key interactive elements.
- Headline font: 'Poppins' for a contemporary, geometric, fashionable look. Note: currently only Google Fonts are supported.
- Body font: 'Inter' for a neutral and readable feel. Note: currently only Google Fonts are supported.
- Use minimalist icons with a slight gradient effect, reflecting the primary color scheme.
- Subtle animations for transitions and interactions, maintaining a modern and engaging user experience.
